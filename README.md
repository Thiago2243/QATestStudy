# QA Automation Study Project / Projeto de Estudos em Automação de Testes QA

---

## Overview / Visão Geral
**English:**  
This project is focused on studying and practicing **Quality Assurance (QA) Automation Testing** concepts and techniques.  
It uses **TypeScript** and **Gherkin (Cucumber)** to define and automate test scenarios in a readable and maintainable way.  

**Português:**  
Este projeto tem como objetivo o **estudo e prática de conceitos de Automação de Testes de Qualidade (QA)**.  
É desenvolvido em **TypeScript**, utilizando **Gherkin (Cucumber)** para definir e automatizar cenários de teste de forma legível e estruturada.  

---

## Tech Stack / Tecnologias Utilizadas
| Component | Descrição / Description |
|------------|------------------------|
| **Language / Linguagem** | TypeScript |
| **Framework** | Cucumber / Gherkin |
| **Test Runner / Executor de Testes** | (e.g., Mocha, Jest, or Cucumber.js) |
| **Data Handling / Tratamento de Dados** | External data sets (JSON, CSV, etc.) / Conjuntos de dados externos (JSON, CSV etc.)
